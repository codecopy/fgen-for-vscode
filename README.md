## 介绍

本项目是专为 `vue-ele-form-generator` 设计的vscode插件, 使其更贴近实际开发, 让大家的操作跟简单方便.

## 使用

- 调出命令输入框, 输入 `fgen: 打开表单设计器` 即可
- 右键, 菜单最底部, 点击 `fgen: 打开表单设计器`
